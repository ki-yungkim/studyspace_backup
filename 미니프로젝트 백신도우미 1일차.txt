#요구사항

	1. 백신 접종 도우미 시스템을 개발하고자 한다.

	2. 어제 접종자 수와 누적 접종자 수를 조회한다.

	3. 일반인은 접종자 조회, 센터 조회, 우선접종 대상자 조회, 백신 2차 접종일 안내, 접종일 알림 기능을 사용할 수 있게 한다.

	4. 관리자는 접종자 수 정보를 변경, 센터 정보 수정을 할 수 있다.

	5. 일반인은 메인 메뉴에서 어제 접종자 수 조회와 누적 접종자 수를 선택하여 정보를 얻을 수 있다. 

	6. 일반인은 메인 메뉴에서 우선접종 대상자 여부 조회를 선택하면  이름, 나이, 직군을 입력하고 대상자일 경우 
	   근처 센터를 조회할 수 있다.

	7. 일반인은 메인메뉴에서 예방접종 센터 조회를 선택해서 필요 정보를 입력하여 조회할 수 있다.

	8. 일반인은 맞은 백신 별 2차 접종 대기 기간을 조회할 수 있고 알림을 받을지 선택할  수 있다.

	9. 일반인은 알림을 받기 위해 등록한 정보를 수정할 수 있다.

#클래스

[DTO]
센터 클래스
- 속성 : 센터명(String), 시설명(String), 우편번호(String), 주소(String), 연락처(String)

멤버 클래스
- 속성 : 이름(String), 연락처(String), 주소(String), 주민번호(String), 
        백신 종류(String), 1차 접종일(String), 2차 접종 예정일(String, 자동 계산), 알림 예정일(String, 자동 계산)  


[Service]
센터 서비스 클래스
	>> 초기 등록(void) : void
	>> 추가 등록(센터명, 시설명, 우편번호, 주소, 연락처) : boolean
	>> 목록에서 제거()
	>> 전체 제거(void) : void
	>> 일부 변경(void) : void - 메서드 내부에서 사용자 입력 받아서 처리
	>> 전체 조회(void) : void
	>> 입력 주소 내 센터 조회(우편번호) : void - 메서드 내부에서 출력
	>> 키워드로 조회(키워드 String) : void - 메서드 내부에서 출력

멤버 서비스 클래스
	>> 초기 등록(void) : void
	>> 추가 등록(이름, 연락처, 주소, 주민번호, 백신 종류, 1차 접종일) : boolean
	>> 목록에서 제거()
	>> 전체 제거(void) : void
	>> 일부 변경(void) : void - 메서드 내부에서 사용자 입력 받아서 처리
	>> 예상 2차 접종일 계산(백신종류, 1차 접종일) : String - 메서드 내부에서 계산 후 등록 메서드로 넘김
	>> 예상 알림일 계산(2차 접종 예정일) : String - 메서드 내부에서 계산 후 등록 메서드로 넘김
	>> 알림

일반 서비스 클래스
	>> 오늘 접종자 조회
	>> 누적 접종자 조회
	>> 우선접종 대상자 식별(이름, 나이, 직업)
	>> 백신별 2차 접종 대기기간 조회



	1. 어제 접종자 조회
	2. 누적 접종자 조회
	3. 우선접종대상자 여부 조회
		> 대상자 X -> 종료
		> 대상자면 근처 센터 조회할래? ->  x선택  -> 종료
						  -> o선택 -> 보여줌

	4. 예방접종 센터 조회
		> 전체조회
		> 메인메뉴 돌아가기  
		> [지역, 센터명, 시설명, 전화번호, 우편번호 등]

	5. 백신별 2차 접종 대기기간 조회
		> 맞은 백신 종류랑 1차 접종일 -> 예상 2차 접종일 출력
							-> 2차 알림 받을래?  -> X 종료
										   -> O 백신 알림기능으로

	6. 백신 알림기능 - 접종받은 백신 종류, 접종일, 이름, 연락처, 주소, 주민번호(식별자) 입력하면
					2차 접종일 3일 전에 입력한 연락처로 2차 접종 예정일, 근처 센터 안내하는 알림 보냄

	7. 내 정보 관리 기능
			- 이름, 주민번호로 본인 확인 -> 등록정보 있음 -> 조회/수정/삭제/메인으로 돌아가기
						-> 등록정보 없음 -> 등록할래? -> x선택 -> 종료
									   -> o선택 -> 백신 알림기능
	8. 관리자 메뉴
		- 관리자 확인 - 접종자 수 변경, 센터 정보 변경,  백신알림기능 신청자들 수정, 삭제 

	9. 프로그램 종료
